> *"Captures the forging of a script from a prompt."*

**SCRIPTFORGE** is a next-generation, interactive infinite worldbuilding engine. Describe a fantasy world once, and the system maintains a living, breathing "World Bible" (lore, maps, factions, characters) that remembers, grows, and generates stories that stay absolutely consistent—forever. 

To top it off, the interface features a **Crazy Hollywood Cinematic Mashup Theme** — blending the magical golden lighting and floating candles of *Harry Potter*, the dark gothic academia and purple stained-glass of *Wednesday Addams*, the cozy whimsy of *Stuart Little* sailing a tiny boat, the eerie shadows of *The Nun* & *Annabelle*, and the tragic grandeur of the sinking *Titanic*!

---

## ✨ Core Engines & Features

- **📜 World Bible Knowledge Graph:** The beating heart of the system. It tracks entities, relationships, timeline events, and world rules in an intricate web, ensuring that your world never contradicts itself.
- **🗺️ Interactive World Atlas:** Explore floating cities on an interactive atmospheric map with glowing, candle-like nodes and deep lore sidebars.
- **👥 Deep Character Engine:** Characters aren't just text. They have specific goals, defining traumas, distinct speech styles, shifting emotional states, and dynamic relationship webs.
- **⚔️ Faction Politics:** Watch powers rise and fall with live alliance webs, ideology tracking, and strength meters.
- **📜 Story Forge:** Generate high-quality literary chronicles, dialogues, prophecies, and character arcs that pull directly from the active World Bible.
- **🛡️ Consistency Checker:** Every generated story is strictly validated. The engine catches logical contradictions (e.g., dead characters speaking, physical sense limits being broken, geographical impossibilities) and logs them in the Integrity dashboard.

## 🚀 How to Run Locally

Since SCRIPTFORGE is a pure frontend application, it's incredibly easy to run!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/stackpilotkulsum/SCRIPTFORGE-Captures-the-forging-of-a-script-from-a-prompt.git
   ```
2. **Navigate into the folder:**
   ```bash
   cd SCRIPTFORGE-Captures-the-forging-of-a-script-from-a-prompt
   ```
3. **Start a local server** (If you have Node.js installed, use `serve`):
   ```bash
   npx serve .
   ```
4. **Open your browser** and navigate to the local address (usually `http://localhost:3000`).
