## ❓ What is SCRIPTFORGE?

**SCRIPTFORGE** is an AI-powered, interactive **Infinite Worldbuilding Engine**. 

Traditional AI story generators suffer from "amnesia"—they forget details, contradict themselves, and lose track of the world's rules. SCRIPTFORGE solves this by taking a single creative prompt and generating a living, breathing **"World Bible"** (a complex Knowledge Graph of lore, maps, factions, and characters). 

This World Bible acts as the absolute ground truth. Every new story, dialogue, or prophecy generated by the engine is strictly validated against this Bible to ensure that the world remembers, grows, and stays logically consistent forever.

---

## 🧠 Machine Learning & Workflow

SCRIPTFORGE operates on an advanced Agentic Pipeline, utilizing multiple specialized engines working in tandem:

1. **World Genesis (The Seed):** 
   The user inputs a prompt. The engine parses this input to seed the initial Knowledge Graph (World Bible), establishing physics, magic systems, political factions, and a 1000-year timeline.
2. **Story Engine (The Generator):** 
   When a user requests a story (e.g., a 'Character Arc' or 'Chronicle'), the Story Engine queries the World Bible. It injects strict contextual data into the prompt—such as a character's deep-seated trauma, current emotional state, and active rivalries—ensuring highly grounded generation.
3. **Consistency Checker (The Validator Agent):** 
   Before a story is finalized, it passes through the Consistency Checker. This agent cross-references the text against the World Bible's rules. If a character uses a magic system that costs them their sight, but the story describes them "looking" at something, the checker flags it as a `CONTRADICTION`, logs it, and calculates a World Integrity Score.

## 🎮 How to Use the Website

1. **Forge Your World:** On the landing page, enter a creative prompt (or click a quickstart option like *"Dying Empire on a Gas Giant"*). Click **Forge This World** and watch the engine initialize.
2. **Explore the Atlas:** Once generated, you'll drop into the World Map. Click on the glowing interactive nodes to uncover deep lore about specific regions, climates, and controlling factions.
3. **Inspect the Lore:** Use the top navigation to browse **Characters** (viewing their sense indicators and traumas), **Factions** (monitoring power bars and alliance webs), and the **Timeline** (filtering 1000 years of history).
4. **Generate Stories:** Navigate to the **Stories** tab. Select a story type (Dialogue, Battle, Prophecy) and a specific character, then hit **Forge Story**.
5. **Monitor Integrity:** Jump to the **Integrity** tab to view your world's overall health score, read the hardcoded world rules, and view the log of contradictions the engine has caught.

## 🚀 How to Run Locally

Since SCRIPTFORGE is a pure frontend application, it's incredibly easy to run!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/stackpilotkulsum/SCRIPTFORGE-Captures-the-forging-of-a-script-from-a-prompt.git
   ```
2. **Navigate into the folder:**
   ```bash
   cd SCRIPTFORGE-Captures-the-forging-of-a-script-from-a-prompt
   ```
3. **Start a local server** (If you have Node.js installed, use `npx serve`):
   ```bash
   npx serve .
   ```
4. **Open your browser** and navigate to `http://localhost:3000`.
